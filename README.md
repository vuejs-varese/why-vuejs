# Why Vue.js?

A non-technical introduction to Vue.js and why you should use it.

## Prerequisites

Make sure to have [**Bun**](https://bun.com/) installed on your system.

## Project Setup

```sh
bun install
```

### Compile and Hot-Reload for Development

```sh
bun run dev
```
